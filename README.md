# Gestionale Karate

Progetto per la gestione di una palestra di Karate, sviluppato seguendo le linee guida di Antigravity.

## Posizione Progetto
`C:\Users\Simone\.gemini\antigravity\scratch\Gestionale Karate`

## Tech Stack
- **Linguaggio**: Python
- **UI**: PySide6
- **Database**: SQLite (SQLAlchemy)
- **Stile**: qt-material

## Setup Environment
1. `python -m venv venv`
2. `.\venv\Scripts\activate`
3. `pip install PySide6 sqlalchemy qt-material`
